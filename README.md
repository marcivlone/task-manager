# Task Manager — менеджер задач

Простое веб-приложение для создания и просмотра списка дел.  
Проект выполнен в рамках обучающей задачи для знакомства с серверным рендерингом и работой с базами данных.

## 🚀 Функционал

- Просмотр списка всех задач
- Добавление новых задач с названием и описанием
- Дата создания проставляется автоматически
- Данные сохраняются в базу данных PostgreSQL

## 🛠️ Стек технологий

- **Backend:** Node.js
- **Шаблонизатор:** Handlebars (серверный рендеринг)
- **База данных:** PostgreSQL
- **Контроль версий:** Git, GitHub

## 💻 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/marcivlone/task-manager.git
   cd task-manager
2. Установите зависимости:
   ```bash
   npm install
4. Настройте базу данных PostgreSQL:

Убедитесь, что PostgreSQL запущен.

Создайте базу данных taskdb.

В файле db.js укажите свои параметры подключения (пароль и т.д.).

4. Запустите сервер:
   ```bash
   node server.js
   
5.Откройте в браузере: http://localhost:3000

## Примечания
Чтобы остановить сервер, нажмите Ctrl + C.

Поле «Название» обязательно для заполнения.

Проект использует серверный рендеринг (Handlebars).
