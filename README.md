# Task Manager — менеджер задач

Простое веб-приложение для создания и просмотра списка дел.  
Проект выполнен в рамках обучающей задачи для знакомства с серверным рендерингом и работой с базами данных.

## 🚀 Функционал

- Просмотр списка всех задач
- Добавление новых задач с названием и описанием
- Дата создания проставляется автоматически
- Данные сохраняются в базу данных PostgreSQL

## 🛠️ Стек технологий

- **Backend:** Node.js
- **Шаблонизатор:** Handlebars (серверный рендеринг)
- **База данных:** PostgreSQL
- **Контроль версий:** Git, GitHub

## 💻 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/marcivlone/task-manager.git
   cd task-manager
2. **Установите зависимости**
    ```bash
    npm install
3. **Настройте базу данных PostgreSQL**
Убедитесь, что PostgreSQL установлен и запущен.

Создайте базу данных с именем taskdb (можно через psql или pgAdmin).

В файле db.js укажите свои параметры подключения:

javascript
const { Pool } = require('pg');

const pool = new Pool({
  user: 'postgres',
  host: 'localhost',
  database: 'taskdb',
  password: 'ваш_пароль',
  port: 5432,
});

module.exports = pool;
4. **Запустите сервер**
bash
node server.js
5. **Откройте в браузере**
Перейдите по адресу: http://localhost:3000

📝 Примечания
Чтобы остановить сервер, нажмите Ctrl + C в терминале.

При добавлении задачи обязательно заполнять поле «Название».

Проект использует серверный рендеринг — страница генерируется на сервере и отправляется в браузер.
